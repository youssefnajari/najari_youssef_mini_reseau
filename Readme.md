L'ESSENTIEL - Mini Réseau Social

Nom et description du projet

L'ESSENTIEL est une plateforme de réseau social minimaliste développée par Youssef Najari. Ce projet simule les fonctionnalités essentielles d'un réseau social moderne avec une interface utilisateur intuitive, responsive et entièrement en français. L'application reproduit l'expérience des réseaux sociaux avec un flux d'actualités, des interactions sociales et un système de messagerie.

Technologies utilisées

- HTML5 : Structure sémantique moderne avec balises adaptées
- CSS3 : Variables CSS, Grid Layout, Flexbox, Media Queries, animations
- JavaScript Vanilla : Gestion des interactions, manipulation du DOM, système dynamique
- Git & GitHub : Gestion de version, déploiement avec GitHub Pages

Fonctionnalités principales

Interface Utilisateur :
- Design moderne avec palette de couleurs cohérente
- Layout responsive s'adaptant à tous les appareils
- Navigation sticky pour un accès facile
- Sidebar interactive avec indicateurs visuels

Fonctionnalités Sociales :
- Système de publications avec images et texte
- Interactions sociales : likes, commentaires, partages, bookmarks
- Notifications en temps réel avec compteurs
- Gestion des demandes d'amis avec acceptation/refus

Messagerie et Communication :
- Messagerie intégrée avec différentes catégories
- Système de commentaires complet avec ajout en temps réel
- Barre de recherche fonctionnelle

Expérience Utilisateur :
- Animations fluides sur les interactions
- Feedback visuel pour toutes les actions
- Design cohérent sur tous les éléments
- Interface entièrement en français

Difficultés rencontrées

1. Layout Grid Complexe :
- Problème : Conteneur de droite mal positionné
- Cause : Valeurs CSS incorrectes et structure HTML mal imbriquée

2. Responsive Design :
- Problème : Interface non adaptative sur mobile
- Cause : Unités absolues et media queries insuffisantes
